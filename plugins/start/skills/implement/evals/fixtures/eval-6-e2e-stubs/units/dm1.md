# Data Types

## Goal
Define data types for the echo API.

## Requirements
- Define EchoRequest type with "message" field (string)
- Define EchoResponse type with "message" and "timestamp" fields
- Export a function createEchoResponse(message) that returns EchoResponse

## Constraints
- Full test coverage
